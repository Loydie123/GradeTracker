Scope of the Study / Project 
“This project focuses on the development of a Student Grade Tracker Mobile Application 
using Kotlin in Android Studio. 
The application is intended for students who want a simple and organized way to record and 
monitor their academic performance. 
The scope of the system includes: 
● Allowing users to add, edit, and delete subjects. 
● Recording grades for quizzes, assignments, and examinations. 
● Applying weighted grading computations for accurate results. 
● Automatically calculating the GPA or Grade Point Average. 
● Tracking academic performance through summaries and simple charts. 
● Providing basic grade prediction based on current scores. 
● Storing all data locally using SQLite through the Room Database. 
● Supporting offline functionality without requiring internet access. 
The project is limited to a single-user mobile application only. 
It does not support cloud storage, online synchronization, or multi-device access. 
The application is designed specifically for Android devices and focuses only on academic 
grade tracking and computation.”
